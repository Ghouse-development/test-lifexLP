# LIFE X LP Design Document

## Overview

株式会社GハウスのFC規格住宅「LIFE X」のランディングページをリニューアル。
THE BASE（thebase-harmony.jp）のスクロール型ワンページ構成を参考に、
エンドユーザー（住宅購入検討者）をメインターゲットとし、FC加盟向け情報も含む。

## Target

- **Primary**: 住宅購入検討者（エンドユーザー）
- **Secondary**: FC加盟検討の工務店・ビルダー

## Tech Stack

- WordPress（既存サイトlifex-house.jpと統一）
- レスポンシブデザイン（モバイルファースト）

### Section 2: 問題提起 — なぜLIFE Xなのか？
THE BASE同様の「問題提起 → 解決」フローで導入。

- 見出し: 「高性能住宅は、本当に高いのか？」
- 業界の常識（高性能＝高価格）を疑問視する問いかけ
- LIFE Xの解決策：規格化で実現した「高性能 × 低コスト」
- 数字で具体化: 「性能はそのまま、価格は○○万円から」
- テキスト量: 150字程度、簡潔に

### Section 3: 選ばれる5つの理由
THE BASE参考: 番号付き（01〜05）＋英語＋日本語の二層見出し＋交互レイアウト

各理由は以下の構成:
- 番号（01〜05）＋英語タイトル＋日本語タイトル
- 写真（大きめ）＋ 説明テキスト（300字以内）
- 具体的な数値で訴求
- 技術用語には必ず平易な言い換えを併記

1. **01 EARTHQUAKE PROOF / 地震に強い家**
   - 耐震等級3（最高等級）＋制振ダンパー搭載
   - 「許容応力度計算」＝建物の強さを1本1本の柱から計算する最も精密な方法
   - 数値: 「震度7相当の揺れにも耐える構造」

2. **02 THERMAL INSULATION / 一年中快適な断熱性能**
   - 断熱等級6 / Ua値0.46
   - 「Ua値0.46」＝北海道基準を超える断熱性能
   - 数値: 「年間光熱費 約○○万円削減」

3. **03 SOLAR ENERGY / 太陽光で光熱費ゼロへ**
   - 最大積載量11.04kW
   - 「月々の電気代が実質ゼロ、さらに売電収入も」
   - 数値: 「月々○○円の売電収入」

4. **04 DESIGN SELECT / 選べる5つのインテリア**
   - プロがコーディネートした5つのテイストから選ぶだけ
   - 「打ち合わせ不要、迷わない」
   - 各テイストのサンプル写真をグリッド表示

5. **05 COST PERFORMANCE / 驚きのコストパフォーマンス**
   - 本体価格1,680万円〜
   - 「高性能住宅がこの価格で手に入る理由」＝規格化によるコスト削減
   - 月々のローン試算: 「月々○○円〜」

## Design Principles — 「超わかりやすい」を実現する

THE BASEの手法を踏襲:

1. **問題提起 → 解決策の流れ**: セクション2で業界の常識に疑問を投げかけ、LIFE Xを解決策として提示
2. **数字で具体化**: 各セクションに必ず具体的な数値を入れる（「約○万円削減」「月々○円」等）
3. **01〜05の番号付きセクション**: 英語＋日本語の二層見出しで視覚的に階層化
4. **技術用語 + 平易な言い換え**: 「許容応力度計算（＝最も精密な強度計算）」のように併記
5. **テキスト量は最小限**: 各理由は300字以内、仕様は20〜40字
6. **図・写真・アイコン中心**: 文字だけで説明しない。必ず視覚要素を伴う
7. **中間CTAの配置**: セクション3の後、セクション5の後にも「資料請求」ボタンを配置

その他:
- THE BASEの構成を参考にしたスクロール型ワンページ
- モバイルファーストのレスポンシブデザイン
- 写真素材を活かした大胆なビジュアル
- ダークネイビー × ゴールドの上質なカラースキーム
- スクロールアニメーション（フェードイン等）で適度な動き
